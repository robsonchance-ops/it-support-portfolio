# Network User Guide

This guide was written for non-technical staff to allow safe, basic interaction with the office network.

## Covers
- Static vs Dynamic IP setup
- Wired and wireless connections
- IoT device onboarding
- IoT management app usage
- Simple troubleshooting steps

## Purpose
Demonstrates:
- Clear technical communication
- End-user support skills
- Real-world IT documentation

This is the type of documentation used in service desk and MSP environments.
