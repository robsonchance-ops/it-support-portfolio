# Smart Office Network – Cisco Packet Tracer

## Overview
A simulated smart office network designed to support office staff, servers, and IoT devices on separate subnets.

## Features
- Multiple VLAN-style subnets
- Static IPs for servers and printers
- DHCP for client devices
- Wireless access point
- IoT server with smart devices

## Key Skills
- IP addressing & subnetting
- DHCP configuration
- Network testing (ping, tracert)
- IoT device connectivity

- `SmartOffice.pkt` – Open with Cisco Packet Tracer
